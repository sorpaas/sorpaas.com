Please direct bugs or suggestions to the issues list on one of our repos ## Installation GSS will go parallel with [Web Workers](http://www.html5rocks.com/en/tutorials/workers/basics/) when the browser supports it. Thus, you need two JS files to get started; `gss.js` & it's helpful `worker.js`. Use the `GSS_CONFIG` object to set worker URL.
