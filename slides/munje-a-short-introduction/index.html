The Munje Programming Language

Munje means
"the universe" in Lojban

The Design

  1. Logical Programming Language
  2. Time-static, side-effects
  3. Simple
  4. Yet, powerful

Advanced Lojban Human-Computer Communication.

Example

	      define factorial
	        factorial 0 1
	        factorial n y
	        (>) n 0
	        (-) n 1 n'
                factorial n' y'
                (*) y' n y

Concepts

  1. Relations (bridi)
  2. Terms (selbri)
  3. Arguments (sumti)
  4. Paragraph (girzu)

Oh, that's the only 4 concepts in Munje.
